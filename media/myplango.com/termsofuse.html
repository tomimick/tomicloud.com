2. Plango Service

With Plango you can setup private group activities on the go. When you invite friends to plans, all invited participants will receive either a push notification (native Plango application users) or an SMS message (new users or Plango mobile web users). SMS invitation contains a URL to a private plan page, which contains the plan details, RSVP actions, and commenting. Recipients are authenticated using the unique URL they receive.

Plango users can establish friend relationships by inviting other Plango users or new users to become their friends. Friend relationship allows sharing plans within the friend graph of the plan organizer.

Plango provides native mobile application, mobile web, and full web interfaces with messaging and calendar functionality for signed up users.
