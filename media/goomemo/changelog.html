List of published versions.

 

Version 2.0.3 - Dec 30, 2010

  • Bug fix: Synching a note containing chars "&" or "<" or ">" now works.
  • Bug fix: Automatically capitalizes sentences during edit.
  • Bug fix: Some synch fixes.
  • Bug fix: When editing in landscape, correctly displays multiple lines.
  • Bug fix: Title shows "+" in the number of notes only when the number exceeds 200.
  • Shows a notification when a note is saved.

Version 2.0.2 - Oct 18, 2010

  • Bug fix: Tags no longer duplicate when saving settings. (Android 2.2? Atleast on HTC Desire.)
  • Bug fix: Soft keyboard is not displayed on startup. (Android 2.2? Atleast on HTC Desire.)

I finally got my own Android 2.2 device and was able to fix the long-standing tag duplication. All worked fine in the 2.2 emulator, oh well.

Version 2.0.1 - June 27, 2010

  • Bug fix: The row of tags did not scroll horizontally on Droid.

Version 2.0 - June 22, 2010

GDocs synchronization plus new features.

  • New feature: 2-way synchronization with a GDocs spreadsheet.
  • New feature: Alphabetical sort.
  • New feature: Vibration on delete can be disabled.
  • Bug fix: Camera problems. Attaching camera pictures crashed Droid. No pictures appeared on HTC Hero.
  • Bug fix: No longer accepts empty tag names.

Why such a long silence after version 1.1?

Honestly, I was waiting for the Android Market to allow me to sell the app for a dollar to have little back of my coding investment. Sadly, that never happened. Being located in Finland, I still can't sell Android apps in Android Market.

Secondly, I still don't have an Android device of my own. Been waiting for the perfect phone with a keyboard to arrive...

Anyway, here's the update finally. Thus there won't be a commercial GooMemo Pro. Long live FREE GooMemo.

Version 1.1 - Sep 13, 2009

Release with essential data exchange functionality.

  • New feature: Data export and import to/from a text file in SD card.
  • New feature: Delete item in context sensitive menu.
  • New feature: Desktop shortcut for quick edit.
  • New feature: Find button on the right of search field.
  • Bug fix: Note truncation in search results was incorrect.

Version 1.0.1 - Sep 9, 2009

First FREE version.

  • Change: Removed GDocs synch functionality.

Version ADC2 - Aug 31, 2009

First version, submitted to Android Developer Contest 2.
